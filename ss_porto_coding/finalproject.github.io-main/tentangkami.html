Apa itu SiBadra ?

SiBadra (Sistem Informasi Berbagi Aduan dan Saran) adalah media bagi Masyarakat Kota Bogor untuk mempermudah dalam menyampaikan pengaduan, saran, dan permintaan layanan kegawatdaruratan kepada Pemerintah Kota Bogor secara real time.
